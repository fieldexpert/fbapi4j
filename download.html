Dependencies

fbapi4j has two dependencies: slf4j 1.5.10 and mime util 2.1. fbapi4j-0.5.jar includes both of these dependencies (slf4j-nop-1.5.10.jar, slf4j-api-1.5.10.jar, and mime-util-2.1.jar). fbapi4j-nodep-0.5.jar does not include any of these dependencies, so you would need to make them available on your classpath if you use this version of the JAR.

Git

You can also clone the project with Git by running:
